# BroodyHen

Plugin for PoeHelper.
Displays a frame above non-incubated items.

Do not need to compile! Just create folder in Plugins/Source/ and drop sources like this:
```sh
.
└── PoeHelper/
    ├── Plugins/
    │   └── Source/
    │       └── BroodyHen/
    │           ├── Main/
    │           │   ├── Core.cs
    │           │   └── Settings.cs
    │           ├── BroodyHen.csproj
    │           └── ...
    └── ...
```
